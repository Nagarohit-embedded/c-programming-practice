# C Programming Practice

This repository contains basic C programs for improving problem-solving and logic-building skills.

## Programs Included

### 1. Spiral Matrix Traversal

Prints a 2D matrix in spiral order.

#### Example Input:

1 2 3
4 5 6
7 8 9

#### Output:

1 2 3 6 9 8 7 4 5


#### Concepts Used:

* Arrays
* Loops
* Boundary conditions
* Problem solving logic

## Purpose

This repository is used to practice C programming fundamentals and improve coding skills.
